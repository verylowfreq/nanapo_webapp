Nanapo Web Controller

USB 7セグディスプレイ「Nanapo (AD7SGPR)」をブラウザから操作します

お使いのブラウザはWeb Serial APIにもWebUSB APIにも対応していません。 PCではChromeまたはEdge、Androidでは Chrome の最新版をお使いください。
